# M1 – Update Army Points

**Actor**: Moderator  
**Preconditions**: Moderator logged in.

## Main Flow:
1. Open “Points Management” panel.
2. Search or browse for squad/unit.
3. Edit points value.
4. Save changes.

## Alternative Flows:
- 3a. If points are invalid, system prompts correction.

## Postconditions:
- Updated points reflected in all future calculations.
