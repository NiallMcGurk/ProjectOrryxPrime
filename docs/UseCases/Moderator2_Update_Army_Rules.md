# M2 – Update Army Rules

**Actor**: Moderator  
**Preconditions**: Moderator logged in.

## Main Flow:
1. Open “Rules Management” panel.
2. Search or browse for faction/squad.
3. Edit or replace rules text.
4. Save changes.

## Alternative Flows:
- 3a. If rule conflicts with system format, validation fails.

## Postconditions:
- Updated rules are available to all users.
