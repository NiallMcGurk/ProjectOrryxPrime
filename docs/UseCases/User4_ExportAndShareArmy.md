# RU4 – Export and Share Army

**Actor**: Regular User  
**Preconditions**: Army exists.

## Main Flow:
1. Open army profile.
2. Select “Export as PDF.”
3. System generates PDF with army, points, and rules.
4. Save or share PDF.

## Alternative Flows:
- 3a. If export fails, retry option appears.

## Postconditions:
- PDF saved or shared.
