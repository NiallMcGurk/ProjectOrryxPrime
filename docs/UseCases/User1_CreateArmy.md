# RU1 – Create an Army

**Actor**: Regular User  
**Preconditions**: User is logged in.

## Main Flow:
1. Select “Create New Army.”
2. Select army name, faction, and points limit.
3. Save army.

## Alternative Flows:
- 3a. If no points limit is entered, default applies.

## Postconditions:
- Army saved to user’s account.
