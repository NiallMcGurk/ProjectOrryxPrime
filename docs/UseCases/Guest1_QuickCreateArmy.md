# G1 – Quickly Create Army

**Actor**: Guest Visitor  
**Preconditions**: App is open; no account required.

## Main Flow:
1. Select “Quick Army Builder.”
2. Choose faction.
3. Add squads using simplified form.
4. System shows total points.

## Alternative Flows:
- 3a. If points exceed limit, show warning.

## Postconditions:
- Army list is viewable but not saved permanently.
